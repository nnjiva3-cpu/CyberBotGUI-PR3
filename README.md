# 🔐 CyberBotGUI - Cybersecurity Awareness Bot (Part 2)

## Description
A WPF GUI chatbot that educates users on cybersecurity topics including passwords, phishing, safe browsing, malware, privacy and scams.

## Features
- GUI built with WPF
- Voice greeting on launch
- ASCII art header
- Keyword recognition for cybersecurity topics
- Random responses using Lists and Dictionaries
- Sentiment detection (worried, frustrated, curious)
- Memory and recall (remembers name and favourite topic)
- Conversation flow (tell me more, give me another tip)
- Error handling for invalid inputs
- Delegates for response handling and message display

## How to Use
1. Run the application
2. Enter your name when prompted
3. Click topic buttons or type questions
4. Type help to see all topics
5. Type bye to exit

## Topics Covered
- Passwords
- Phishing
- Safe Browsing
- Malware
- Privacy
- Scams
- Two Factor Authentication

## Project Structure
- MainWindow.xaml - GUI layout
- MainWindow.xaml.cs - All logic, responses and delegates
